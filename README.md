# ****DQ3風バトルシミュレータ****

## **概要**

DQ3に出てくる敵や魔法,武器を用いたバトルシミュレータです。

**出現する敵**

lev = 1 : 'スライム','いっかくウサギ','ホイミスライム','さまようよろい','キングスライム','メタルスライム','はぐれメタル'

lev = 2 : 'さまようよろい','ベホマスライム','キングスライム','カンダタこぶん','トロール','はぐれメタル','メタルキング'

lev = 3 : 'ベホマスライム','カンダタ','キングスライム','トロール','ボストロール','はぐれメタル','メタルキング'

## **操作方法**

1.ユーザ名入力

2.敵の強さを入力

3.バトルコマンドを入力する

4.どちらかのHPが0になるまで続ける
